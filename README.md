# inTheAir
2D airplane game with Three.js

## Control
- Arrow keys to move
- Space key to shoot
- B key to drop bomb

## Story
- Era: In the WW2
- War Zone: Europe
- Alliance: America, Westen Europe
- Enemy: Germany, Japan
- Ego: France Air Force

## Style
- Genre: Shooting Game
- Ego's Moving Direction: Down -> Top
- Items: Enhance the Bullet size, count and moving.
- Enemy: Shoot bullets
